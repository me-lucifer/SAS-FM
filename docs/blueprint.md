# **App Name**: SAS Fleet Manager

## Core Features:

- Dashboard Overview: Provides a summary of key fleet metrics (e.g., active vehicles, upcoming maintenance, fuel consumption) using charts and data tables. The tool can identify anomalies in the data.
- Vehicle Management: Allows adding, editing, and removing vehicle profiles with details like model, registration, and assigned driver.
- Driver Management: Enables adding, editing, and removing driver profiles with details like contact information and license details.
- Fuel Queue Tracking: Monitors fuel levels across the fleet, sends alert when low.
- Odometer Tracking: Records and displays vehicle odometer readings over time to detect possible vehicle misue
- Maintenance Scheduling: Schedules and tracks vehicle maintenance, and reminds of service timing based on date or kilometers traveled
- Reports Generation: Generates customizable reports on fleet performance (e.g., fuel consumption, maintenance costs) within a selected date range.

## Style Guidelines:

- Primary color: Steel blue (#6699CC) for a professional, trustworthy feel.
- Background color: Light gray (#F5F7FA) for a clean and unobtrusive background.
- Accent color: Sea green (#73C2BE) to provide subtle contrast and highlight interactive elements.
- Font pairing: 'Inter' (sans-serif) for body text and 'PT Sans' (sans-serif) for headlines to ensure readability and a modern look.
- Left sidebar for primary navigation and top bar for page title and date-range picker. Desktop-first, responsive design.
- Consistent use of clear, simple icons throughout the interface to represent different features and actions.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
